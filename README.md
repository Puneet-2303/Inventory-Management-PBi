## Project Overview

An interactive Power BI dashboard for inventory management in an e-grocery business. Helps management monitor available stock, inventory value, expiry risk, reorder needs, and supplier performance. Data is fetched live from GitHub CSV files.

## Key Features

- Real-time stock availability tracking
- Inventory value monitoring
- Expiry risk analysis to reduce waste
- Reorder alerts for low-stock items
- Supplier performance evaluation
- Live data sourced from GitHub

1. Download dashboard.pbix
2. Open in Power BI Desktop
3. Dashboard connects live to GitHub for data
